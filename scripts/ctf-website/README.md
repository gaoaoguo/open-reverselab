# CTF Website Scripts

Web CTF 分析脚本。

## 主要脚本

- `kb_router.py` — 知识库路由器，按信号查找相关技术
- `cve_lookup.py` — CVE 查询
- `cve_graph.py` — CVE 关联图谱
- `cve_chain_planner.py` — CVE 利用链规划
- `fingerprint_cve_pipeline.py` — 指纹→CVE 自动化流水线
- `http_probe.py` — HTTP 探测
- `ctf_intake.py` — CTF 题目导入
- `ctf_ai_next.py` — AI 下一步建议
- `ctf_new_challenge.ps1` — 新题目初始化
- `ctf_toolcheck.ps1` — 工具可用性检查
